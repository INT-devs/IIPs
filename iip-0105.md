# IIP-105: Height in Coinbase

## Abstract

This IIP documents INTcoin's adoption of BIP-34, requiring the block
height to be encoded in the coinbase transaction's scriptSig. The
mechanism is implemented without modification and has been mandatory
since genesis.

## Specification

Implements [BIP-34](https://github.com/bitcoin/bips/blob/master/bip-0034.mediawiki)
without divergence.

The coinbase scriptSig must begin with a push of the block height as
a CScriptNum. Negative heights are rejected. Height must match the
actual chain height of the block.
