# IIP-208: Mempool P2P Message
## Abstract
Documents INTcoin's adoption of BIP-35 mempool message, allowing peers to request the contents of each other's mempool. Implemented without modification.
## Specification
Implements [BIP-35](https://github.com/bitcoin/bips/blob/master/bip-0035.mediawiki) without divergence.
