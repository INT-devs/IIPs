# IIP-410: Multisig HD Derivation Path
## Abstract
Documents INTcoin's adoption of BIP-48 multisig HD derivation paths for ISMS (IIP-0409) multisig wallet coordination. Path format: `m/48'/2210'/account'/script_type'`.
## Specification
Implements [BIP-48](https://github.com/bitcoin/bips/blob/master/bip-0048.mediawiki) without divergence in path structure. Coin type 2210 per IIP-0103. Derivation uses IIP-0301 (SHA3-256) instead of BIP-32 (HMAC-SHA512).
## Reference Implementation
`src/wallet/hdchain.h`, `src/wallet/walletmanager.cpp`
