# IIP-413: DNS Payment Instructions
## Abstract
Documents INTcoin's planned adoption of BIP-353 DNS-based payment instructions, allowing users to receive payments via human-readable names (e.g. `user@domain.com`) resolved through DNS TXT records.
## Status
**Final — verified present in the tree as of 2026-04-24.** The library implementation in
`src/wallet/dnspayments.{h,cpp}` has been in the tree; the missing
piece was a user-facing entry point. The `resolvedns` RPC added in
`src/rpc/wallet.cpp` now surfaces `ResolvePaymentAddress` so
operators can look up `user@domain` handles against DNS and retrieve
the parsed payment URI (address + optional amount + label +
DNSSEC-validation status).
### RPC commands
- `resolvedns <address> [require_dnssec]`
## Reference Implementation
`src/wallet/dnspayments.h`, `src/wallet/dnspayments.cpp`,
