# IIP-407: Bech32 Encoding
## Abstract
Documents INTcoin's adoption of BIP-173 Bech32 encoding for addresses and extended keys. The encoding algorithm is unchanged; INTcoin uses network-specific HRPs defined in IIP-0400 (`int`, `tint`, `sint`, `rint`) and extended key HRPs defined in IIP-0301 (`ixpub`, `ixprv`).
## Specification
Implements [BIP-173](https://github.com/bitcoin/bips/blob/master/bip-0173.mediawiki) encoding without divergence. HRPs and address semantics are defined in IIP-0400.
