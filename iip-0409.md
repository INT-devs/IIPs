# IIP-409: BSMS Multisig Setup
## Abstract
Documents INTcoin's adoption of BIP-129 Bitcoin Secure Multisig Setup (BSMS) for coordinating multisig wallet creation between multiple signers. Adapted for Dilithium5 keys and IIP-0301 derivation paths.
## Specification
Implements [BIP-129](https://github.com/bitcoin/bips/blob/master/bip-0129.mediawiki) with the following divergences:
- **Signer key records** contain 2,592-byte Dilithium5 public keys.
- **Derivation paths** use IIP-0301 (BIP-48 style: `m/48'/2210'/account'/script_type'`).
- **Descriptor format** uses IIP-0408 output descriptors.
- **Encryption token** protects key records during exchange.
### Workflow
1. Coordinator initialises session with M-of-N parameters.
2. Each signer exports their key record (encrypted with session token).
3. Coordinator collects all key records and generates the final multisig descriptor.
4. All signers verify the first derived address matches.
