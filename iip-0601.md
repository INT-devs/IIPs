# IIP-601: Block Template Protocol
## Abstract
Documents INTcoin's adoption of BIP-22 getblocktemplate for mining pool and standalone miner communication. Miners request block templates from a running node and submit solved blocks. Implemented without modification to the RPC interface.
## Specification
Implements [BIP-22](https://github.com/bitcoin/bips/blob/master/bip-0022.mediawiki) without divergence. The `getblocktemplate` and `submitblock` RPCs follow the standard protocol.
