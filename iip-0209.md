# IIP-209: Bloom Filters (Disabled)
## Abstract
Documents INTcoin's decision to disable BIP-37 bloom filter support. Compact block filters (IIP-0203) provide a superior alternative for light client filtering without the privacy and DoS issues inherent to bloom filters.
## Specification
BIP-37 bloom filter messages (FILTERLOAD, FILTERADD, FILTERCLEAR, MERKLEBLOCK) are recognized but not serviced. Peers requesting filtered blocks receive no response. The `bloomfilter` network permission flag can override this for whitelisted peers.
## Rationale
Bloom filters leak wallet addresses to the serving node and allow cheap DoS via expensive filter matching. IIP-0203 compact block filters replace this functionality with better privacy and bounded server cost.
