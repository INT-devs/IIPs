# IIP-211: Reject Messages (Deprecated)
## Abstract
Documents that BIP-61 reject messages are recognized but deprecated in INTcoin. Nodes do not send REJECT messages. The message type is retained for protocol compatibility with peers that may still send them.
## Specification
Implements [BIP-61](https://github.com/bitcoin/bips/blob/master/bip-0061.mediawiki) as deprecated. REJECT messages received are logged and ignored. No REJECT messages are sent.
