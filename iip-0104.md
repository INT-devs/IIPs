# IIP-104: Version Bits Deployments

## Abstract

This IIP documents INTcoin's adoption of BIP-9 version bits for
coordinating soft fork activations. The mechanism is implemented
without modification.

## Specification

Implements [BIP-9](https://github.com/bitcoin/bips/blob/master/bip-0009.mediawiki)
without divergence.

Deployment state machine: DEFINED → STARTED → LOCKED_IN → ACTIVE (or FAILED).

### Active deployments

| Name | Bit | Activation | Threshold | IIP |
|------|-----|-----------|-----------|-----|
| TESTDUMMY | 0 | Testing only | 90% | — |
| EXTENDED_OPRETURN | 4 | Height 1000 (mainnet) | 90% | IIP-47 |

### Reserved bits

Bits 1–3 are reserved for future deployments (not yet configured).
