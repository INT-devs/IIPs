# IIP-411: Anti-Fee-Sniping
## Abstract
Documents INTcoin's adoption of BIP-326 anti-fee-sniping via nLockTime. Wallet-created transactions set nLockTime to the current block height, discouraging miners from re-mining old blocks to claim high-fee transactions. Implemented without modification.
## Specification
Implements [BIP-326](https://github.com/bitcoin/bips/blob/master/bip-0326.mediawiki) without divergence. The wallet sets `nLockTime = current_height` for all new transactions unless an explicit locktime is provided.
